# AI_agent
# Telegram Career Assistant

Автоматизированный AI-агент на базе **n8n**, **GigaChat-2** и **Cloud.ru**, который ежедневно агрегирует новости из профильных Telegram-каналов (ODS, geoML, Mashin Learning TV) и еженедельно анализирует вакансии с **HH.ru**, выявляя топ-15 востребованных навыков.  
Все данные поступают через **FetchRSS** и **публичное API HH**, обрабатываются двумя вызовами GigaChat-2 (извлечение + агрегация) и отправляются в приватный Telegram-канал.  

Проект помогает специалисту в Machine Learning принимать обоснованные карьерные решения — без выдумок, только актуальные данные.
